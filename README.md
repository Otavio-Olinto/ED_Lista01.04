# ED_Lista01.04
## Quantidade de negativos num vetor

Este desafio consiste em criar uma aplicação, em Java, que exiba a quantidade de números negativos que ha num vetor de inteiros.

```
Requisitos do projeto:

1. Arquitetura MVC;
2. A busca deve ser realizada por uma função recursiva;
3. O vetor e seu tamanho devem ser passados como parâmetro da função;
4. Declarar, via comentário, a condição de parada da função.
```
